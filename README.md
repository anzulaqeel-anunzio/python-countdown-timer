# Countdown Timer CLI

A simple yet effective countdown timer for your terminal. Supports various time formats and provides a visual update in-place.

## Features

*   **Flexible Input**: Accepts seconds (`10`), minutes (`1m`), or combined formats (`1h30m`).
*   **Visual Feedback**: Updates the timer on the same line to keep your terminal clean.
*   **Audio Alert**: Emits a system beep when the time is up.

## Usage

```bash
python run_timer.py [time] [options]
```

### Examples

**1. 30 Seconds**
```bash
python run_timer.py 30
```

**2. 5 Minutes (using flag)**
```bash
python run_timer.py 5 -m
```

**3. Complex Duration (1 minute 30 seconds)**
```bash
python run_timer.py 1m30s
```

**4. Pomodoro (25 minutes)**
```bash
python run_timer.py 25m
```

## Requirements

*   Python 3.x
